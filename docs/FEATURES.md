# Функциональные возможности Isolate

## Основные функции

### 1. Автоматический подбор стратегий

**Режим Turbo (быстрый)**
- Проверяет топ-5 популярных стратегий для региона
- Время: ~10 секунд
- Использует кэш успешных стратегий

**Режим Deep (глубокий)**
- Полный перебор всех доступных стратегий
- Время: 2-3 минуты
- Включает экзотические методы обхода

### 2. Диагностика блокировок

Автоматическое определение типа DPI:

| Тип | Признак | Рекомендуемая стратегия |
|-----|---------|------------------------|
| Timeout | Нет ответа | Фрагментация пакетов |
| Instant RST | Мгновенный сброс | TTL trick, Fake TCP |
| MITM | Неверный сертификат | VLESS прокси |

### 3. Block QUIC

- Блокировка UDP 443 через Windows Firewall
- Принуждает браузеры использовать TCP/TLS
- Включено по умолчанию (можно отключить)

### 4. Panic Button (Аварийный сброс)

Доступен из системного трея даже при зависании UI:
- Убивает все процессы winws/sing-box
- Сбрасывает Winsock: `netsh winsock reset`
- Сбрасывает IP-стек: `netsh int ip reset`
- Опционально: перезапуск сетевого адаптера

### 5. Пользовательские сервисы

Добавление своих доменов для тестирования:
1. Ввод домена (например, `example.com`)
2. Автоматический резолв IP
3. Добавление в список тестирования
4. Проверка доступности при оптимизации

### 6. Импорт конфигураций

Поддержка форматов:
- `vless://...` — VLESS прокси
- `vmess://...` — VMess прокси
- Drag-n-drop или вставка из буфера

### 7. Автообновление

- Списки доменов (hostlist) обновляются автоматически
- Стратегии обновляются без переустановки программы
- Источник: GitHub releases

## Дополнительные функции

### Dual-Stack (IPv4/IPv6)

- Тестирование по обоим протоколам
- Автоматическое определение, где DPI слабее
- Опция "Отключить IPv6" как стратегия

### Логирование

- Детальные логи в `%APPDATA%/Isolate/logs/`
- Уровни: Error, Warn, Info, Debug
- Ротация логов (хранение 7 дней)

### Portable Mode

Запуск с флагом `--portable`:
- Все данные в папке с программой
- Не требует установки
- Удобно для USB-флешки

### Silent Mode

Запуск с флагом `--silent`:
- Без UI, только иконка в трее
- Автоприменение последней успешной стратегии
- Для автозапуска с Windows
