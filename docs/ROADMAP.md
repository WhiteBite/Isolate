# Roadmap Isolate

## Фаза 1: MVP (v0.1.0)

### Ядро
- [ ] Базовая структура Tauri + SvelteKit
- [ ] Process Manager (запуск/остановка winws)
- [ ] Загрузка стратегий из YAML
- [ ] Последовательное тестирование стратегий
- [ ] Применение выбранной стратегии

### UI
- [ ] Главный экран со статусом
- [ ] Кнопка "Оптимизировать"
- [ ] Список стратегий с результатами
- [ ] System tray с базовыми действиями

### Инфраструктура
- [ ] Bundling winws + WinDivert
- [ ] Логирование в файл
- [ ] Базовые настройки (SQLite)

---

## Фаза 2: Core Features (v0.2.0)

### Функционал
- [ ] Режимы Turbo / Deep
- [ ] Диагностика типа блокировки
- [ ] Block QUIC (firewall rule)
- [ ] Panic Button
- [ ] Поддержка Sing-box (VLESS)

### UI
- [ ] Визуализация процесса тестирования
- [ ] Настройки приложения
- [ ] Добавление пользовательских доменов
- [ ] Импорт vless:// ссылок

### Оптимизация
- [ ] Параллельное тестирование parallel_safe стратегий
- [ ] Кэширование успешных стратегий
- [ ] Таймауты и graceful shutdown
